# Syllabus-Based YouTube Video Recommender

This project recommends relevant YouTube videos based on the input syllabus text provided by the user.

## How to Use

1. Install Python and the required libraries:
   - `nltk`
   - `google-api-python-client`
2. Obtain a YouTube Data API key from the Google Developer Console.
3. Insert your API key into the `main.py` file at the specified place.
4. Run the `main.py` script.
5. Paste your syllabus text when prompted.
6. The program will output YouTube video links corresponding to each topic/unit.

## Dependencies

- Python
- nltk
- google-api-python-client

## Security Note

- Never share your API keys publicly.
- Use the `.gitignore` file to exclude sensitive files like API key files from version control.
